Коли-небудь пробували відправити секретне повідомлення комусь не користуючись послугами пошти? Ви можете використовувати газету, щоб розповісти комусь свій секрет. Навіть якщо хтось знайде ваше повідомлення, легко відмахнутися і сказати що це параноя і теорія змови. Один з найпростіших способів заховати ваше секретне повідомлення це використовувати заголовні літери. Давайте знайдемо кілька таких секретних повідомлень.

Дана частка тексту. Зберіть усі великі літери в одне слово в тому порядку як вони зустрічаються в шматку тексту.

Наприклад: текст = "How are you? Eh, ok. Low or Lower? Ohhh.", якщо ми зберемо усі великі букви, то отримаємо повідомлення "HELLO".

Вхідні дані: Текст як рядок (юнікод).

Вихідні дані: Секретне повідомлення як рядок або порожній рядок.

Приклад: 

{% if interpreter.slug == "js-node" %}findMessage{% else %}find_message{% endif %}("How are you? Eh, ok. Low or Lower? Ohhh.") == "HELLO"
{% if interpreter.slug == "js-node" %}findMessage{% else %}find_message{% endif %}("hello world!") == ""

Як це використовуэеться: Це проста вправа по роботі з рядками: ітерація, читання і об'єднання.

Передумова: 0 < len(text) ≤ 1000
all(ch in string.printable for ch in text)
