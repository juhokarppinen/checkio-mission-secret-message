"Πού θα κρύψει ένας σοφός άνθρωπος ένα φύλλο; Στο δάσος. Όμως, τι θα κάνει αν δεν υπάρχει δάσος; Θα δημιουργήσει ένα, ώστε να μπορέσει να το κρύψει... "
-- Gilbert Keith Chesterton

Προσπάθησες ποτέ να στείλεις ένα μυστικό μήνυμα σε κάποιον χωρίς να χρησιμοποιήσεις το ταχυδρομείο; Θα μπορούσες να χρησιμοποιήσεις τις εφημερίδες για να μεταδώσεις το μυστικό σου μήνυμα. Ακόμα και αν κάποιος ανακαλύψει το μήνυμα, εύκολα θα ισχυριστείς ότι είναι παρανοϊκός και φαντάζεται θεωρίες συνομωσίας! Ένας εύκολος τρόπος να κρύψεις ένα μήνυμα είναι να χρησιμοποιήσεις τα κεφαλαία γράμματα ενός κειμένου. Ας δούμε μερικά παραδείγματα.

Δίνεται κάποιο κείμενο. Ένωσε όλα του τα κεφαλαία γράμματα σε μια λέξη, με τη σειρά που εμφανίζονται μέσα στο κείμενο.

Για παράδειγμα: κείμενο = "How are you? Eh, ok. Low or Lower? Ohhh.", αν ενώσουμε όλα του τα κεφαλαία γράμματα θα σχηματίσουμε τη λέξη "HELLO".

Είσοδος: Ένα κείμενο με τη μορφή string (unicode).

Έξοδος: Το μυστικό μήνυμα με τη μορφή string ή το κενό string.

Παράδειγμα: 

{% if interpreter.slug == "js-node" %}findMessage{% else %}find_message{% endif %}("How are you? Eh, ok. Low or Lower? Ohhh.") == "HELLO"
{% if interpreter.slug == "js-node" %}findMessage{% else %}find_message{% endif %}("hello world!") == ""

Πώς χρησιμοποιείται: Είναι μια απλή άσκηση για τις βασικές λειτουργίες των strings: προσπέλαση, αναγνώριση και ένωση.

Προϋποθέσεις: 0 < len(text) ≤ 1000
all(ch in string.printable for ch in text)
